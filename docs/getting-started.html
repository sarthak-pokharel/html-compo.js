Getting Started

Learn HTML-COMPO.js in 5 minutes

1

Include the Library

Add HTML-COMPO.js to your HTML page. That's it - no build tools, no complex setup!

<!DOCTYPE html>
<html>
<head>
  <title>My First HTML-COMPO App</title>
  <script src="https://cdn.jsdelivr.net/npm/html-compo@latest/src/main.js"></script>
</head>
<body>
  <!-- Your content goes here -->
</body>
</html>

3

Add Component Attributes

Make your components configurable by using the compo-attrs attribute to specify which HTML attributes should be available as variables.

<html-components>
  <user-profile compo-attrs="name,role,department">
    <div style="background: #f8fafc; padding: 15px; border-radius: 8px; border-left: 4px solid #667eea;">
      <h3>@{name}</h3>
      <p><strong>Role:</strong> @{role}</p>
      <p><strong>Department:</strong> @{department}</p>
    </div>
  </user-profile>
</html-components>

<user-profile name="Alice Johnson" role="Senior Developer" department="Engineering"></user-profile>
<user-profile name="Bob Smith" role="Product Manager" department="Product"></user-profile>

4

Set Default Values

Provide default values for attributes using the attribute=defaultValue syntax in compo-attrs.

<html-components>
  <status-badge compo-attrs="status=active,color=#4caf50">
    <span style="background: @{color}; color: white; padding: 4px 12px; border-radius: 12px; font-size: 0.8rem;">
      @{status}
    </span>
  </status-badge>
</html-components>

<status-badge></status-badge> <!-- Uses defaults -->
<status-badge status="inactive" color="#f44336"></status-badge>
<status-badge status="pending" color="#ff9800"></status-badge>

5

Interactive Components

Add interactivity by using component references and the global htmlCompo API.

<html-components>
  <click-counter compo-attrs="count=0">
    <div style="text-align: center; padding: 20px; border: 2px solid #e2e8f0; border-radius: 10px;">
      <h3>Counter: @{count}</h3>
      <button onclick="incrementCounter('@{self.componentName}')" 
              style="background: #667eea; color: white; border: none; padding: 10px 20px; border-radius: 5px; cursor: pointer;">
        Click me!
      </button>
    </div>
  </click-counter>
</html-components>

<click-counter ref="myCounter"></click-counter>

<script>
function incrementCounter(componentName) {
  const counter = htmlCompo.getComponent('myCounter');
  if (counter) {
    const currentCount = parseInt(counter.attr('count')) + 1;
    counter.attr('count', currentCount.toString());
  }
}
</script>

Build Something Amazing!

Now you know the basics! Here are some ideas for what you can build:

  • Form Components: Input fields, buttons, validation messages
  • Layout Components: Cards, modals, navigation menus
  • Data Components: Tables, lists, charts
  • Interactive Widgets: Calendars, sliders, tabs
<!-- Example: A complete card component -->
<html-components>
  <info-card compo-attrs="title,type=info,actions=false">
    <div style="border: 1px solid #e2e8f0; border-radius: 10px; overflow: hidden; background: white;">
      <div style="background: @{type === 'success' ? '#4caf50' : type === 'warning' ? '#ff9800' : '#2196f3'}; 
                  color: white; padding: 15px;">
        <h3 style="margin: 0;">@{title}</h3>
      </div>
      <div style="padding: 20px;">
        @{self.data}
      </div>
    </div>
  </info-card>
</html-components>
