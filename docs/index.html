HTML-Compo.js

Reuse the html components! With html-compo, you can reuse the html components. The library is superfast and performant! The html-components tag is used as components defination container. As an alternative h-c tag can be used instead of long html-components tag

user-data Jhon user-data user-data Harry user-data html-components user-data div User Name : b*@{self.data}b div user-data html-components
User Name: Jhon
User Name: Harry


You can use compo-attrs attribute to watch specific attrubutes while using components
user-detail name="Jhon" contactnum="0682936273" description="Jhon is a very versatile employee"user-detail user-detail name="Harry" contactnum="082665623" description="Jhon is a very smart employee"user-detail html-components user-detail compo-attrs="name, contactnum,description" User Name: *@{name} br/ Contact Number: *@{contactnum} br/ Description: *@{description} br/ br/ user-detail html-components
